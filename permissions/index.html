What are Permissions?

        Permissions are a basic system that allows you to easily assign certain roles custom abilities and permissions.
Permissions run on inheritance, meaning that each role will inherit the permissions of its pervious role! Base -> DJ -> ExtraRole -> ModRole -> ExtraRole2 -> Admin (Will have all perms)
For example, if the Base permission set (everyone) has a permission, DJ, ExtraRole, ModRole and ExtraRole2 will also have that permission. Any permissions that ModRole has is also applied to ExtraRole2 but not DJ or ExtraRole. This is because ExtraRole2 is higher than ModRole however ExtraRole and DJ are not. Admin role will get EVERY permission node, this is also the case with any role with the discord Administrator permission!
For example, if you wanted to give the ExtraRole the permissions notlewd, loop, viewCurve, snipe, ripEmote, flushify and export, you would do:
daz perms extrarole notlewd,loop,viewCurve,snipe,ripEmote,flushify,export 
Please note that this sets the permissions and not adds/deletes.
