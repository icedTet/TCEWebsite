Music

nad play <Item>

nad play mousou express, nad play LINK
nad play $$playlist_Key, nad play YoutubePlaylistLink

Adds the specified <Item> to queue or plays it if there is nothing in the queue

Music

nad queue #

nad queue 5 OR nad queue

returns up to the first 20 items from the queue starting from the number given, or 0

Music

nad np

nad np

returns info about the song currently playing

Music

nad remove <pos>

nad remove 1, nad remove all

removes the <Pos>th song from the queue, or all if <Pos> is all. Will not work if user has the 'NoRemoveMusic' role

Music

nad vs

vote skips for the song currently playing (must have a majority of non bots in the vc voting yes)

Music

nad loop

nad loop

Toggles Loop Mode ON or OFF (Loop Mode basically just copies the songs to the back of the queue every time they play)

Admin Settings

nad levelreward LEVEL ROLEID

nad levelreward 5 59128051 OR
nad levelreward 5 none

Sets a role to be given once somebody achieves that level. Or clears that role given when roleid is NONE
(Requires Role named Admin)

Admin Settings

nad reactionrole

reactionrole create ChannelID MessageID Emoji RoleID

Creates a reaction role, where when a user reacts, they gain the role; unreacting removes the role

Admin Settings

nad setmessageevent

nad setmessageevent join CHANNELID MESSAGE

Sets up custom messages for when a person Joins the server(Dm message also avalible), leaves the server, or levels up

Admin Settings

nad delevelreward LEVEL ROLEID

nad delevelreward 5 59128051 OR
nad delevelreward 5 none

Sets a role to be removed once somebody achieves that level. Or clears that role removed when roleid is NONE. Basically the reverse of levelreward.
(Requires Role named Admin)

DO NOT USE THIS IN A MAIN CHANNEL. THIS CAN PING ALOT OF PEOPLE!!! USE THIS IN A PRIVATE SPAM CHANNEL!!!
